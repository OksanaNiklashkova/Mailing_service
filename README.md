# Mailing_service

Этот проект предоставляет возможности для организации рассылок писем в рамках развития бизнеса.

## Описание
В проекте представлено веб-приложение, обеспечивающее работу с почтовыми рассылками. 
Чтобы воспользоваться возможностями нашего сервиса, в верхней части страницы нажмите на кнопку "Регистрация" или авторизуйтесь на сайте.
Авторизованные пользователи могут создавать свои рассылки и составлять списки их получателей. Сервис предоставляет возможности для анализа успешности рассылок.
Пользователь, создавший рассылку, автоматически получает статус ее владельца и в любой момент может отредактировать: добавить или убрать реквизиты, изменить содержание описание, остановить, возобновить и повторить.

## Точка входа
Для перехода в веб-приложение запустите команду `python manage.py runserver`  и нажмите на ссылку в консоли `http://127.0.0.1:8000/`

## Установка

Для установки и запуска проекта необходимо выполнить следующие шаги:

1.  **Клонируйте репозиторий:**

    ```
    https://github.com/OksanaNiklashkova/Mailing_service.git
    ```

2.  **Перейдите в папку проекта:**

    ```
    cd Mailing_service
    ```

3.  **Установите зависимости с помощью Poetry:**

    ```
    poetry install
    ```
4. **Запустите команду запуска сервера, чтобы перейти в веб-приложение**
    ```
    python manage.py runserver
    ```

## Использование

Примеры использования:

Зарегистрируйтесь в веб-приложении, указав свою электронную почту и пароль. Пароль должен быть не короче 8 символов и не состоять только из цифр.

~~~
Авторизуйтесь на сайте. 
Перейдите на страницу "Получатели рассылки" и создайте список получателей.
На странице "Сообщения" составьте сообщение новой рассылки.
Переключитесь на страницу "Рассылки" и создайте новую рассылку, добавив в нее сообщение, получателей, заполнив дату и время начала и завершения.
На странице сервиса "Pассылки" выберете созданную рассылку и нажмите "Отправить". Письма с вашим сообщением будут отправлены всем получателям, указанным в списке.
Просмотреть успешность рассылки можно во вкладке "Попытки рассылки".

Отправка активных рассылок возможна из терминала с использованием кастомной команды: 
python manage.py send_mailing
Все рассылки авторизованного пользователя, имеющие статус "создана" или "запущена" будут отправлены и выведен краткий отчет об успешности рассылок.
~~~
